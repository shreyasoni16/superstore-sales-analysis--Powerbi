# superstore-sales-analysis--Powerbi

1. Superstore Sales Dashboard
This project is a Power BI dashboard analyzing the Superstore dataset to provide insights on sales performance, profit trends, regional performance, and product category analysis. It demonstrates data analysis, visualization, and business intelligence skills.

2. Purpose
The purpose of this project is to analyze the Superstore sales dataset and create an interactive Power BI dashboard that helps businesses understand sales performance, profit trends, and product category performance. It allows users to identify top-performing regions, profitable products, and customer segments, enabling data-driven business decisions.
